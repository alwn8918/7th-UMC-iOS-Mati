## Commit Convention

| 아이콘 | 이름 | 설명                     |
| ---- | --- | ------------------------ |
| 🎉 | Begin | 초기 프로젝트 세팅 |
| ✨ | Feat | 새로운 기능 추가 |
| 🐛 | Fix | 버그 수정 |
| 💄 | Design | UI, 스타일 관련 추가 및 수정 |
| ♻️ | Refactor | 코드 리팩토링 |
| 📝 | Docs | 문서 파일 추가 및 수정 |
| 🔥 | Remove | 파일 및 코드 삭제 |
| 🔀 | Merge | 브랜치 병합 |
| 💡 | Add | 주석 추가 및 수정 |
